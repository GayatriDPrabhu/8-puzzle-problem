8 Puzzle Problem

#To compile the program, do following from the command prompt:

$ javac PuzzleProblem.java


#To run the program, do following from the command prompt:

$ java PuzzleProblem <algorithm name>

  algorithm name = 'dfs' for Depth-first search
		 = 'ids' for Iterative deepening search
		 = 'a*'  for A* search

  eg: java PuzzleProblem ids


#To enter input (initial state), any of the following two formats can be used -

  1) * 1 3 4 2 5 7 8 6

  2) * 1 3
     4 2 5
     7 8 6
